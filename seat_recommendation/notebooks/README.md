# Jupyter ノートブック

このディレクトリには、データ探索や実験用のJupyterノートブックを配置できます。

## 推奨ノートブック

1. **data_exploration.ipynb**: データの可視化と統計分析
2. **model_comparison.ipynb**: 複数モデルの性能比較
3. **feature_importance.ipynb**: 特徴量重要度の分析
4. **user_clustering.ipynb**: ユーザーのクラスタリング分析

## Jupyter Labの起動

```bash
pip install jupyterlab
jupyter lab
```
